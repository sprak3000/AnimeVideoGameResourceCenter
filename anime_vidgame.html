Anime Video Game Resource Center

Welcome! This page is designed to provide you with a useful collection of links and information pertaining to anime that has been turned into video games. If you have any information or links you would like to add, please email me.

WHAT'S NEW (4/21/97): Added new Godzilla ROM info. Added link to new Quake patch.

And now a plea for help. Between school and work, I don't have much time to play some of the GameBoy games on this page. So, if you have a review, some codes, strategy hints, etc. for these games, please send them to me so I can post more information on these games.
